<h1>Step 6: Literal Nodes</h1>

<p>Now let's set a <span class="term">literal node</span> for the
<span class="code">=alice</span><span class="code">&lt;#email&gt;</span> attribute. To do so, we
can now add a literal arc that establishes a literal node
under the attribute node.</p>

<p>Literal nodes hold the actual data in an XDI graph,
for example <span class="code">"alice@email.com"</span>. All JSON data types are supported.
The address of the literal node in the graph is <span class="code">=alice</span><span class="code">&lt;#email&gt;</span>&.</p>
