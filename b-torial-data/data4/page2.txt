<h1>Step 2: HTTP POST Example</h1>

<p>This is a simple example of a request message and response message,
sent over HTTP from an XDI client to an XDI server.</p>

<p>The standard HTTP <span class="term">Accept</span> and <span class="term">Content-Type</span> headers can be used
to indicate the XDI serialization format to use:</p>

<ul>
<li><span class="term">XDI DISPLAY</span> format: text/xdi</li>
<li><span class="term">JXD</span> format: application/xdi+json</li>
</ul>

<p>In the next steps, we will look at the exact structure of an XDI message.</p>
